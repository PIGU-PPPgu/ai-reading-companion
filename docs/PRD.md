# AI学习伴侣 -- 产品方案 v4.0

> 全教材覆盖，自定义内容，NotebookLM式音频生成，从名著阅读到全科学习

## 一、产品定位

**一句话**: 上传任意教材/书籍/资料，AI帮你拆解知识、生成学习计划、互动讨论、播客式回顾。家长全程可见。

**不是什么**:
- 不是电子书阅读器
- 不是AI摘要工具
- 不是题库
- 不限于语文/名著

**是什么**: 一个**全科学习家教**。任何教材、任何书籍、任何资料(包括网上电子书)，AI都能帮你拆解、带读、检测理解。像 NotebookLM + DeepTutor 的结合体，但面向K12、有家长端、有教学设计。

用户可以上传或导入:

- EPUB/MOBI电子书
- 网上电子课外书(输入URL自动抓取)
- 学习笔记/讲义
- 网页文章/博客
- 老师发的复习资料

### 2.3 内容类型适配

| 内容类型 | 拆解方式 | 带学策略 | 测评重点 |
|----------|----------|----------|----------|
| 文学名著 | 章节/人物/主题/写作手法 | 沉浸式带读+讨论 | 理解深度+文学鉴赏 |
| 数学教材 | 知识点/公式/例题/变式 | 概念讲解+练中学 | 解题能力+举一反三 |
| 英语教材 | 词汇/语法/课文/听力 | 情景化学习+跟读 | 词汇+语法+阅读理解 |
| 科学教材 | 概念/实验/公式/应用 | 探究式学习+思维导图 | 概念理解+实验分析 |
| 历史/地理 | 时间线/事件/人物/因果 | 故事化叙事+时间线 | 记忆+分析+论述 |
| 课外电子书 | 自动识别内容类型 | 自动匹配策略 | 自动生成 |
| 自定义资料 | 自动识别 | 自动匹配 | 自动生成 |

## 三、核心功能

### 3.1 内容拆解引擎
- 自动识别内容类型
- 结构化拆解(章节/知识点/人物/公式等)
- 生成知识图谱
- 考试考点标注

### 3.2 个性化学习计划
- 根据年级/进度/时间生成计划
- 每日学习任务
- 自适应节奏

### 3.3 苏格拉底式对话
- 不直接给答案，引导思考
- 根据内容类型调整对话风格

### 3.4 NotebookLM式音频生成
- **Audio Overview**: 上传任何内容，一键生成5-10分钟播客式回顾
- 双人对话形式(Host+Guest)，自然有趣
- 支持中文多角色TTS
- 可以在通勤/运动时听
- 家长版: 每周学习内容音频总结

### 3.5 测评系统
- 多题型(选择/填空/简答/计算/论述)
- 对标中考/高考
- 间隔重复，薄弱点强化

### 3.6 家长端
- 学习进度看板
- 理解度报告
- 讨论指引
- 周报/月报

### 3.7 Flashcard + Mindmap (CuFlow灵感)
- 自动从学习材料生成闪卡
- 自动生成思维导图
- 间隔重复记忆

## 五、技术架构

```
用户层
  学生端(小程序)    家长端(小程序)    教师端(Web)

API Gateway

DeepTutor基座(改造)
  Core: Agent + RAG + Memory + Knowledge Graph

  插件层(我们开发):
    content-analyzer    -- 内容类型识别+结构化拆解
    learning-guide      -- 个性化学习计划
    socratic-dialog     -- 苏格拉底对话
    assessment          -- 多题型测评+间隔重复
    parent-report       -- 家长报告/周报
    audio-companion     -- NotebookLM式播客生成
    content-manager     -- 内容上传/URL导入/管理
    flashcard           -- 闪卡生成+复习
    mindmap             -- 思维导图生成

数据层
  ChinaTextbook教材库 + 电子书库
  Neo4j知识图谱
  用户数据(PG+Redis)
  音频存储(S3/OSS)
```
