# AI学习伴侣 -- 产品方案 v3.0

> 全教材覆盖，自定义内容，从名著阅读到全科学习

## 一、产品定位

**一句话**: AI带着学生学透任意教材/书籍，拆解知识、生成图谱、互动讨论、家长可见。

**不是什么**:
- 不是电子书阅读器
- 不是AI摘要工具
- 不是题库/练习册
- 不限于语文/名著

**是什么**: 一个**全科学习家教** -- 任何教材、任何书籍、任何资料，AI都能帮你拆解、带读、检测理解。而且这个家教是AI，24小时在线，只要几十块/月。

### 2.2 自定义内容

用户可以上传:
- 任意PDF教材/教辅
- 课外书籍文本
- 学习笔记/讲义
- 网页文章/博客

### 2.3 内容类型适配

不同类型的内容，AI采用不同的"带学"策略:

| 内容类型 | 拆解方式 | 带学策略 | 测评重点 |
|----------|----------|----------|----------|
| 文学名著 | 章节/人物/主题/写作手法 | 沉浸式带读+讨论 | 理解深度+文学鉴赏 |
| 数学教材 | 知识点/公式/例题/变式 | 概念讲解+练中学 | 解题能力+举一反三 |
| 英语教材 | 词汇/语法/课文/听力 | 情景化学习+跟读 | 词汇+语法+阅读理解 |
| 科学教材 | 概念/实验/公式/应用 | 探究式学习+思维导图 | 概念理解+实验分析 |
| 历史/地理 | 时间线/事件/人物/因果 | 故事化叙事+时间线 | 记忆+分析+论述 |
| 自定义内容 | 自动识别内容类型 | 自动匹配策略 | 自动生成 |

## 五、核心差异: 内容智能适配

产品核心不只是"阅读"，而是**根据内容类型自动调整教学策略**。

### 拆解引擎 v2: 内容感知

输入任意内容后:
1. 自动识别内容类型(文学/数学/科学/语言/社科/自定义)
2. 根据类型选择对应的拆解模板
3. 调用LLM进行结构化提取
4. 生成类型适配的知识图谱
5. 对标考试要求标注考点

### 一个例子: 数学 vs 语文

**数学(七年级 一元一次方程)**:
- 拆解: 概念定义 -> 等式性质 -> 解方程步骤 -> 应用题类型
- 知识图谱: 前置知识(有理数) -> 核心概念(方程) -> 后续知识(二元一次)
- 带学: 先概念理解，再例题精讲，再变式练习
- 测评: 基础题 + 应用题 + 拓展题

**语文(骆驼祥子)**:
- 拆解: 章节结构 -> 人物图谱 -> 主题分析 -> 写作手法 -> 中考考点
- 知识图谱: 三起三落曲线 -> 人物关系 -> 主题关系
- 带学: 每日阅读+讨论题+理解检测
- 测评: 情节理解 + 人物分析 + 主题论述
