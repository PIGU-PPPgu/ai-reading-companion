# AI Reading Companion

AI阅读家教 - 基于DeepTutor的K12名著精读平台

## 项目定位

面向中小学生的AI名著精读家教。不是电子书阅读器，不是题库，是一个"阅读家教"。

## 技术架构

- 基座: [DeepTutor](https://github.com/HKUDS/DeepTutor) (Apache-2.0)
- 内容源: [ChinaTextbook](https://github.com/TapXWorld/ChinaTextbook)
- LLM: GLM-5 / DeepSeek
- TTS: 硅基流动

## 核心插件

1. 拆书引擎 - 书籍结构化拆解(章节/人物/知识点/考点)
2. 带读引擎 - 个性化阅读计划生成
3. 苏格拉底对话 - 引导式提问
4. 测评系统 - 理解度检测(对标中考)
5. 家长报告引擎 - 进度/理解度/周报
6. 音频伴侣 - 播客式回顾
